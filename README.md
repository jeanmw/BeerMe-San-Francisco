# BeerMe-SF
Beer Database for San Francisco

BeerMe San Francisco is a beer app that helps users discover and recall their favorite beers at local stores.

[Link to project hosted on Heroku]()

## Technologies Used

*Bootstrap
*JQuery
*MongoDB




## Existing Features

*Search beers
*Delete beers




## Planned Features

*Bigger Database
*Images
*Maps
*Location Edits
